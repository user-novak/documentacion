Laravel Jetstream es un kit de inicio de aplicación bellamente diseñado para Laravel y proporciona el punto de partida perfecto para su próxima aplicación de Laravel. Jetstream proporciona la implementación para el inicio de sesión, registro, verificación de correo electrónico, autenticación de dos factores, administración de sesiones, API a través de Laravel Sanctum y funciones opcionales de administración de equipos.

Tailwind CSS es un framework CSS de bajo nivel altamente personalizable que le permite crear diseños personalizados eliminando los estilos de componentes pre-definidos que necesitarías sobre-escribir en todo caso

AdminLTE AdminLTE es un panel de administración para Bootstrap creado por el estudio Almsaeed. Es una solución de código abierto basada en un diseño modular que permite una construcción y personalización sencillas

Funciones:

Llamado de Seeders: Esta función invoca a los Seeders para su ejecución, producto de esta ejecución es la creación de registros en sus respectivas tablas, de esta forma llenamos la base de datos con valores de prueba.

Creacion de los atributos: Esta función genera datos falsos en los atributos transcritos, los atributos restantes no se usan en esta fucnión por el hecho de usar datos reales y no falsos.
